Django With Vue (You write this)

  • CRUD: {% url 'vue_blog' %} uses the template vue_blog.html which includes the file blog.vue which will contain the meat of this assignment

The idea is to include vue code in the tempalate 'blog/blog.vue' which is inserted into the html template 'blog/vue_blog.html'. You need to make a single page vue application that will list the posts, allow the user to create a new post, edit an existing post or delete a post. To show the post detail/body, make a modal popup display this information. So the files you are concerned with are under blog/templates/blog/:

  • base.html is the base html file that contains the html header for css and js imports
  • vue_blog.html which extends base.html and contains the html
  • base.vue is the base template for vue. It defines
  • blog.vue which extends base.vue and contains your code
The API endpoints are listed under the REST Endpoints heading, and if you call them within a web browser, django rest api will present you with a nice page where you can try them out.

Edit the static/css/staff_test.css, or include stylesheets in base.html to make the page nice. You don't need to use django static files, you can just load them from the internet if you don't understand the django template static command.
